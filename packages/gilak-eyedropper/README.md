# gilak-eyedropper

A React eyedropper color picker component for Gilak Studio.

## Installation

```bash
npm install gilak-eyedropper
# or
pnpm add gilak-eyedropper
```

## Usage

```tsx
import { GilakEyedrop } from 'gilak-eyedropper'

function App() {
  return <GilakEyedrop />
}
```
