# gilak-canvas

A React canvas component for Gilak Studio.

## Installation

```bash
npm install gilak-canvas
# or
pnpm add gilak-canvas
```

## Usage

```tsx
import { GilakCanvas } from 'gilak-canvas'

function App() {
  return <GilakCanvas />
}
```
